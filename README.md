# KEEEB
Making a mechanical keyboard with LED lighting.
This keyboard has a 67 key layout, similar to a 60% keyboard.

I made this project, because I wanted a mechanical keyboard, and to learn how to design pcbs. I felt this would be a challenging project that would make me something I want to use daily.
